# Bika Alfa - Portofolio Website

A modern, responsive, and minimalist portfolio website built with clean HTML, CSS, and Vanilla JavaScript. Features a sleek dark mode aesthetic with premium animations and scroll-triggered effects.

## 🚀 Features

- **Personalized Branding**: Fully customized with name, bio, and authentic social links.
- **Responsive Design**: Optimized for mobile, tablet, and desktop views.
- **Dynamic Components**: Footer and Navbar are loaded dynamically to ensure consistency across all pages.
- **Scroll Animations**: Smooth fade-in effects that trigger as you explore the site.
- **Premium Aesthetics**: High-resolution custom artwork and a cohesive dark color palette.

## 📁 Project Structure

- `index.html`: Home page with hero section and social links.
- `about.html`: Personalized biography and skills.
- `resume.html`: Professional experience and educational background.
- `projects.html`: Showcase of recent work with premium mockups.
- `contact.html`: Functional contact form and information.
- `assets/`: Images, icons, and fonts used across the site.
- `components/`: Reusable HTML components (Navbar, Footer).
- `css/`: Custom styling for all sections.
- `js/`: Core logic and scroll animations.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, grid/flexbox layouts, and animations.
- **JavaScript**: Manual DOM manipulation and Intersection Observer API for animations.

## 🚀 Cara Menjalankan di Localhost

Agar fitur loading Navbar dan Footer berjalan lancar (menghindari error CORS), Anda bisa menjalankannya lewat terminal dengan perintah:

1. Buka terminal di folder project ini.
2. Jalankan perintah berikut:
   ```bash
   npx serve .
   ```
3. Buka link yang muncul (biasanya `http://localhost:3000`).
